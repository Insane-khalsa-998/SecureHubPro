Instructions
To Encode:
  1. Enter your secret message
  2. Upload a PNG image
  3. Enter a password or generate one
  4. Click "Encode Message"
  5. Download the encoded image
  6. Save the password - you'll need it to decode!
To Decode:
  1. Upload an encoded PNG image
  2. Enter the correct password
  3. Click "Decode Message"
  4. The hidden message will appear
Tip: Only use PNG images for best results
